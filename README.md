# Introduction to Logistic Regression

This repository contains all materials used for the *Introduction to Logistic Regression* class, delivered as part of the specialist skills training offered by SEDarc DTP.

The session provides an accessible introduction to fitting logistic regression models in R, combining key theoretical concepts with practical, hands-on examples. You will learn how to fit and interpret simple and multiple logistic regression models for binary outcomes (e.g., *yes/no*), as well as multinomial logistic regression for categorical outcomes with more than two levels and ordinal logistic regression for ordered categorical outcomes. The session also covers the key assumptions of logistic regression and explains how to interpret model coefficients in terms of log odds, odds ratios, and probabilities, including how to convert between these different metrics.

The class materials are contained in the HTML file (and its source R Markdown file). For access to a recording of the session, please contact the SEDarc team.
